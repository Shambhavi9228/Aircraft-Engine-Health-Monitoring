# AI-Based Aircraft Engine Health Monitoring System

## Overview
This project predicts the Remaining Useful Life (RUL) of aircraft engines using LSTM deep learning models.

## Technologies Used
- Python
- TensorFlow
- Keras
- Pandas
- NumPy
- Matplotlib

## Dataset
NASA C-MAPSS Turbofan Engine Dataset

## Features
- Engine Health Monitoring
- RUL Prediction
- Predictive Maintenance
